# Outcome Dealer OS — Premium Internal Portal Design Handoff

This folder contains the full visual mockup reference set for the Outcome Dealer OS internal portal redesign.

## Design Target
Outcome Dealer OS should be rebuilt to match the quality level of these mockups: premium dark metal, cinematic automotive, enterprise SaaS, high-density but organized, polished enough to feel like a flagship dealership operating system.

## Files

- `mockups/01_control_center.png` — Main dashboard / Control Center
- `mockups/02_leads_command_center.png` — Leads page
- `mockups/03_inventory_command.png` — Inventory page
- `mockups/04_customer_360.png` — Customer / Lead detail page
- `mockups/05_deal_desk.png` — Deal Desk / desking page
- `mockups/06_finance_center.png` — Finance / lender management page
- `mockups/07_calendar_execution.png` — Calendar and task execution page
- `mockups/08_analytics_reports.png` — Reports and analytics page
- `mockups/09_ai_copilot.png` — AI Copilot page
- `mockups/10_settings_admin.png` — Settings and admin page
- `copilot/IMPLEMENT_DESIGN_PROMPT.md` — Copy/paste prompt for GitHub Copilot

## Implementation Rule
Treat these images as the design source of truth. Existing app functionality should be preserved, but the UI should be refactored to match this visual system.
